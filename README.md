📌 About the Project
This is a single-page awareness website created for InAmigos Foundation, a Central Government–licensed NGO headquartered in Bilaspur, Chhattisgarh. The webpage is designed to spread awareness about the NGO's ongoing projects, social impact, and initiatives — and to inspire visitors to volunteer, donate, or share the mission.
Built as Task 1 of the InAmigos internship programme using only HTML & CSS (no frameworks).
